Sketch Control

        
Ctrl/Cmd+Enter
Run the sketch

        
Esc
Stop the sketch / cancel a pending wire / exit wire mode

        
Ctrl/Cmd+Shift+R
Reset everything — code, components, wires & state. (Click the Reset button's dropdown arrow to reset just the code or just the wiring.)


        
Wiring & Components

        
W
Toggle wire drawing mode

        
Ctrl/Cmd+Shift+⌫
Clear all wires

        
Delete/Backspace
Remove the selected component


        
Navigating the Board

        
Space+drag
Pan around the breadboard

        
middle-click+drag
Pan around the breadboard

        
scroll
Zoom in / out toward the cursor

        
Ctrl/Cmd++
Zoom in

        
Ctrl/Cmd+−
Zoom out

        
Ctrl/Cmd+0
Reset zoom to 100%


        
Code Editor

        
Tab
Indent the current line


        
Shortcuts that use letter keys (like W) are disabled while you're typing in the code editor or the Serial Monitor input, so they won't interrupt what you're writing.
